Achievements

  • 1887-1888: Developed the Alternating Current (AC) system.
  • 1888: Invented the AC induction motor.
  • 1891: Created the Tesla coil.
  • 1896: Helped build the Niagara Falls hydroelectric power plant.
  • 1897-1898: Developed wireless remote control technology.
  • 1890s: Conducted experiments with X-rays and wireless communication.
