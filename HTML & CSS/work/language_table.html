Definition/Description List

        HTML
            - A markup language used to structure content on the web.
      
        CSS
            - A stylesheet language used to describe the presentation of a document
            written in HTML.
      
        JavaScript
            - A programming language used to create dynamic and interactive effects
            within web browsers. It can be used both for frontend and backend
            development.
      
        Python
            - A versatile programming language often used for backend web
            development, automation, data analysis, and machine learning.
